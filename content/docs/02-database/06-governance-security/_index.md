데이터가 기업의 핵심 전략 자산으로 부상함에 따라, 데이터를 체계적으로 관리하고 보호하는 거버넌스 및 보안 체계의 중요성이 높아지고 있다. 개인정보 보호법·정보통신망법 등 규제 강화와 데이터 침해 사고의 증가는 데이터 거버넌스와 보안을 IT 조직의 핵심 과제로 만들었다.

데이터 거버넌스는 데이터 표준화·품질 관리·아키텍처 체계를 통해 전사 데이터의 일관성과 신뢰성을 확보하는 관리 체계이다. 데이터베이스 보안은 접근 제어·암호화·비식별화로 DB에 저장된 민감 데이터를 무단 접근과 유출로부터 보호한다.

이 단원에서는 공공·민간 부문의 데이터 표준화 체계와 품질 관리 프로세스, 그리고 DAC·MAC·RBAC 접근 제어 모델과 TDE·마스킹·비식별화 기술을 다룬다.
