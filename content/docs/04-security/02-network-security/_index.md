시스템 및 네트워크 보안은 인터넷 경계와 내부망을 보호하기 위한 방화벽·IDS/IPS·WAF·VPN 등 핵심 보안 장비와 접근 통제 기술을 다루며, 외부 위협 차단부터 안전한 원격 접속까지 네트워크 전 계층의 보안 체계를 다룬다.  
각 토픽은 기술 원리·비교 분석·실무 적용 관점에서 구성되어 정보관리기술사·컴퓨터시스템응용기술사 시험과 실무 보안 설계 모두에 활용할 수 있다.

## 학습 토픽

| 토픽 | 핵심 내용 | 보안 영역 |
|---|---|---|
| **방화벽·IDS·WAF·VPN** | 패킷 필터링·NGFW, IDS/IPS 오탐/미탐, WAF OWASP, IPsec·SSL VPN | 경계 보안, 침입 탐지 |
| **접근 통제 모델** | MAC·DAC·RBAC·ABAC, Zero Trust, 최소 권한 원칙 | 접근 제어, 권한 관리 |
| **보안 아키텍처** | DMZ, 심층 방어, 마이크로세그멘테이션, SASE | 네트워크 설계, 구역 분리 |
| **보안 모니터링** | SIEM, SOC, 로그 분석, 이상 탐지, 보안 오케스트레이션 | 탐지·대응, 가시성 확보 |
