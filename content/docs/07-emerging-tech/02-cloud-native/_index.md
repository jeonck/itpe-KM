클라우드 컴퓨팅은 IaaS·PaaS·SaaS 서비스 모델부터 컨테이너·마이크로서비스·서버리스로 대표되는 클라우드 네이티브 아키텍처까지 진화하며 IT 인프라의 표준으로 자리잡았다. 기술사 시험에서는 서비스·배포 모델 비교, 컨테이너 오케스트레이션, DevOps·CI/CD 파이프라인, 비용 최적화 전략이 핵심 출제 범위이므로 각 계층의 책임 경계와 설계 원칙을 체계적으로 이해하는 것이 중요하다.
