title: 주요 응용 알고리즘
weight: 3
---

실전에서 자주 활용되는 정렬, 그래프 탐색, 최소 신장 트리, 문자열 매칭 알고리즘의 설계 원리와 시간복잡도, 선택 기준을 다룬다.

| 토픽 | 주요 내용 |
|---|---|
| [정렬 알고리즘](sorting-algorithms) | 버블·선택·삽입(O(N²)) 기초 정렬, 퀵·병합·힙(O(N log N)) 고속 정렬, 계수·기수 특수 정렬 및 전체 비교 |
| [그래프 탐색 및 최단 경로](graph-traversal-shortest) | BFS·DFS 탐색 원리, 다익스트라·벨만-포드·플로이드-워샬 최단 경로 알고리즘 비교 |
| [최소 신장 트리 (MST)](minimum-spanning-tree) | 크루스칼·프림 알고리즘 동작 원리 및 유니온-파인드 자료구조, 네트워크 최적화 적용 |
| [문자열 매칭 알고리즘](string-matching) | 나이브·KMP·보이어-무어·라빈-카프 알고리즘 원리, 전처리 기법 및 시간복잡도 비교 |
